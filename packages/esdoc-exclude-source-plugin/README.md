# ESDoc Exclude Source Plugin
## Install
```bash
npm install @itsjamie/esdoc-exclude-source-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "@itsjamie/esdoc-exclude-source-plugin"}
  ]
}
```
