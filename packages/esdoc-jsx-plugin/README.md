# ESDoc JSX Plugin
## Install
```bash
npm install @itsjamie/esdoc-jsx-plugin
```

## Config
```json
{
    "source": "./src",
    "destination": "./doc",
    "plugins": [
        {
            "name": "@itsjamie/esdoc-jsx-plugin", 
            "option": {
                "enable": true
            }
        }
    ]
}
```

- `enable` is default `true`
